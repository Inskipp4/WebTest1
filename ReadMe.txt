Ресурс тестирования по вопросам дисциплины «Технологии проектирования программного продукта».
В системе реализована система авторизации и учета пользователя.
Для каждого пользователя выводится результаты попыток прохождения тестирования.
Всего в систему загружено 10 тестовых вопросов, каждый новый тест выбираются 5 случайных вопросов ответы на вопрос перемешиваются.
За основу разработки взят материл: https://habr.com/ru/post/346340/
Использованы стили Bootstrap
Запуск осуществляется из модуля main.py, после чего ресурс доступен по адресу: http://127.0.0.1:8000/
